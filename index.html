Refugee Network Diagram

How do refugees move? One way to visualize movement is on a geographic map. Another way is a network. Each connection represents refugees moving from one country to another. We focus on the 5 largest refugee crises, in Afghanistan, Nigeria, Somalia, South Sudan, and Syria, represented by the larger dots, or nodes. Each smaller node is a country to which refugees are fleeing. The width of each link corresponds to the number of refugees moving in a year, and the length corresponds to measurements of political stability. Countries that differ in stability will be pushed apart, countries similar in stability will cluster together.


Explore how refugees move from the 5 largest ongoing refugee crises. Modify settings and see how the network reacts. Hover over a node to see the country name, and drag any part to move the network around. Start by clicking any button.
